Time Action

This example shows how to use the testing.time utility to show time information for building a target.

Files:

  • jamroot.jam
  • main_cpp.pro

Our jamroot.jam specifies the target we build and the testing.time declaration to time the target we build: 

import testing ;

make main.cpp : main_cpp.pro : @do-something ;

time main.time : main.cpp ;

actions do-something
{
    sleep 2 && echo "$(<)" > "$(<)"
}

Import the time rule from the testing module.

The target we are timing.

This target records the time to build the main.cpp target.

This simple action waits and then copies the source to target.

Building the example yields:
